# Mount Images

Place mount template images here. The frontend loads them by filename from this folder.  
Images should be landscape-oriented JPEGs — **600×400 px** is ideal.

Custom mount images uploaded by the DM are stored in `backend/uploads/mounts/` and served automatically — you don't need to add those here.

---
